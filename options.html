    Keyboad Shortcut

    By default, Chrome does not come with a keyboard shortcut to open Bookmark Manager. Regardless of the reason for this, it is understandable that some would like this feature present. Fortunately, as of Chrome v22.x, it is now possible to set up keyboard shortcuts to trigger extensions, which means with this extension a keyboard shortcut can be created to open the Bookmark Manager. Here's how:

    1. Copy and paste the following URL into the address bar/omnibox: chrome://chrome/extensions/ (Options > Tools > Extensions).

    2. Scroll to the very bottom of the page and click the link "Configure Commands".

    3. When the list appears, next to "Open Bookmark Manager" enter your desired keyboard shortcut. A favourite, commonly used in other browsers, is Ctrl+B.

    4. Hit OK and that's it! Best of all, using your shortcut key will also load your specified bookmarks folder, just like clicking the icon.
